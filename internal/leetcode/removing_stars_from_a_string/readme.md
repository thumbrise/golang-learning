# 2390. Removing Stars From a String

Вам дана строка `s`, которая содержит звездочки `*`.

В одной операции вы можете:
- выбрать звездочку в `s`;
- удалить ближайший незвездный символ слева от нее, а также саму звездочку.

Верните строку после всех операций.

**Примечание:**
- Входная строка будет сгенерирована так, что операция всегда возможна.
- Можно доказать, что результирующая строка всегда будет уникальной.

**Example 1:**

Input: s = "leet**cod*e"

Output: "lecoe"

Explanation: Выполняем удаления слева направо:
- Ближайший символ к 1й звездочке - 't' в "leet**cod*e". s становится "lee*cod*e".
- Ближайший символ к 2й звездочке - 'e' в "lee*cod*e". s становится "lecod*e".
- Ближайший символ к 3й звездочке - 'd' в "lecod*e". s становится "lecoe".
Больше нет звездочек, возвращаем "lecoe".

**Example 2:**

Input: s = "erase*****"
Output: ""
Explanation: Вся строка удаляется, возвращаем пустую строку.

**Ограничения:**

- 1 <= s.length <= 105
- s состоит из строчных английских букв и звездочек `*`.
- Операция выше может быть выполнена на s.
